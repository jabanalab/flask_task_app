# Flask Task Management App

A simple task management application built with Flask, SQLite, and Tailwind CSS.

## Features
- User Registration and Login
- Task Dashboard (CRUD operations)
- API-based backend
- Responsive UI with Tailwind CSS

## Setup Instructions
1.  **Install Dependencies**:
    ```bash
    pip install flask flask-sqlalchemy flask-login flask-bcrypt
    ```
2.  **Run the Application**:
    ```bash
    python app.py
    ```
3.  **Access the App**:
    Open your browser and go to `http://127.0.0.1:5000`.

## Project Structure
- `app.py`: Main Flask application with models and API routes.
- `templates/`: HTML templates (login, register, dashboard).
- `SRS_Document.md`: Detailed Software Requirements Specification.
- `instance/tasks.db`: SQLite database (created on first run).
